# Генератор паролей НГУЭУ
Cервис удобной и быстрой генерации паролей с гибкой настройкой

## Доступ к сервису
https://xx-arteeem-xx-learn-is-lab-1-f460.twc1.net/

## Для разработчиков
This template should help get you started developing with Vue 3 in Vite.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## Для DevOps 
Сервис развернут с помощью **Timeweb App Platform**. При коммите в ветку main производится автоматический деплой приложения.
